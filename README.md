## 技术栈

* vue.js + elementui + axios + jquery + 三剑客

## 页面组成

* 主页面：轮播图 + 显示商品小组件
* 搜索页面：标签云 + 搜索框 + 显示商品小组件 ，可按照名称模糊搜索，也可按照标签搜索，点击标签即可搜索该标签内容
* 上传商品页面：上传商品，可上传图片
* 联系管理员页面 ：跳到[我的博客](http://www.jxdgogogo.xyz) 
* 登录：账号 + 密码 + 
* 注册：可发送邮箱验证码
* 个人界面：显示个人信息、头像等,可修改
* 订单页面：显示账户所有订单
* 收货地址页面：显示所有收获地址
* 商品简介页面：显示商品大图，加入购物车
* 购物车页面：提供新建订单功能
* 订单支付页面：使用沙箱支付宝支付，支付完跳回订单页面
